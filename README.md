## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/shahdshaker/shopping_app.git
   ```
2. Navigate to the project:
   ```bash
   cd shopping_app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
